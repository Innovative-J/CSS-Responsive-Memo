Flex box

Flex box is described as a one dimension layout model that uses properties to change and align elements on its main and cross axis to make it responsive and dynamic.

Sample text.

Flex direction- row, sets items once in a flex container to be adjusted on its main axis.

selector example{
                    flex-direction:row;
                }

Flex direction- -row-reverse, sets items once in a flex container to be adjusted on its main axis starting on a opposite direction.

selector example{
                    flex-direction:row-reverse;
                }

Positioning

Positioningsets how and where the element is positioned in an doc with various properties.

Absolute, the element is removed from a normal document flow and is positioned relative to its nearest ancestor. It's values can be determined by its top, right, bottom and left values.

selector example{
                position:absolute;
                top:10px;
                right:20px;
                bottom:30px;
                left:40px;
            }

Relative, the element is positioned from it's normal document flow and it is relative to itself. It's values can be determined by its top, right, bottom and left values.

selector example{
                position:relative;
                top:10px;
                right:20px;
                bottom:30px;
                left:40px;
            }

Display

Display, defines how an element should be displayed as well as the layout and behavior within the doc flow

Block, block elements generates a line break before and after an element forcing other elements into a new line

selector example{
                display:block;
            }

Inline, generates a inline box that does not create a break before or after an element, as well as not forcing any elements into a new line. 

selector example{
                    display:inline;
     }

Fonts

Font, sets various font properties that correlate to its font family, size, style and various others to create the ideal style for a all or specific font.

Font-family, this property sets or prioritizes one or multiple fonts.

selector example{
                font-family:sans-serif;
     }
     // This property sets the font-family style.//

Font-size, this sets the size of the font.

selector example{
                font-size:1em;
     }
     //This property sets the size to 1em which is the equivalent of 16px.//

The box model properties

The box model is essentailly a box that wraps every HTML element.

Padding, reffers to the space between the content and border which is set on all four sides at once. 

.example {
        padding: 10px 50px 30px 0px;
    }
        //The shorthand starts by top is 10px, right is 50px, bottom is 30px, and left is 0px.//

Border, sets the elements border. The border itself can have a style and color.

.example {
        border: 4px dashed rgba(211, 220, 50, .6);
    }
    // theh shorthand is the border being 4px, while being dashed and using a rgba to specify a color.//
